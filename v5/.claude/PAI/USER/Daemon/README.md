# Daemon

**Purpose:** Your public digital representation — a shareable profile of what you're working on, thinking about, reading, and building.

**What lives here:** Output from the Daemon system, which reads your TELOS, Knowledge, Projects, and other PAI sources and writes a curated public-facing profile. The Daemon decides what's safe to share and what stays private; the artifacts it produces (profile pages, current-focus snippets, reading lists) land here for review or publishing.

**How it gets populated:** By the Daemon skill automatically, on-demand or on a schedule. The skill reads your private PAI context, applies its scrub rules, and writes the public profile here. You can edit the output before publishing if you want finer control.

**Sample state for fresh installs:** Empty / Just this README. Real content appears as you use PAI.
