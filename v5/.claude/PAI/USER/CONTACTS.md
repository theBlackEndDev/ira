# Contacts

> People the DA should know about — short entries, relational context.

## Format

Each entry: name, relationship, context, last touch (optional), handle/email (optional).

```
### Name
**Relationship:** collaborator / friend / client / family / other
**Context:** one-line why they matter
**Handle:** @username / email (if relevant)
**Last touch:** YYYY-MM-DD (optional)
```
